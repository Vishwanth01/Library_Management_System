# 📚 Library Management System

## 🧾 Overview

The **Library Management System (LMS)** is a software application designed to manage and automate the daily operations of a library. It helps librarians efficiently handle book records, user registrations, borrowing/returning processes, and inventory tracking.

---

## 🚀 Features

* 📖 Book Management
  Add, update, delete, and search for books.

* 👤 User Management
  Register and manage library members.

* 🔄 Issue & Return System
  Track borrowed books and due dates.

* ⏰ Fine Calculation
  Automatically calculate overdue fines.

* 🔍 Search Functionality
  Search books by title, author, or category.

* 📊 Reports
  Generate reports on issued books, users, and inventory.

---

## 📌 Usage

* Admin can add/update/remove books.
* Users can register and borrow books.
* System tracks due dates and calculates fines.
* Search and filter books easily.

---

## 🔐 Roles

* **Admin**

  * Manage books and users
  * View reports

* **User**

  * Search books
  * Borrow/return books
